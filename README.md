# Mellow Munch

Food-forward snack subscription brand mockup — multi-page responsive site with AI-generated product imagery.

## Pages

- `/` — Home
- `/themes.html` — Theme comparison
- `/how-it-works.html` — Flow + FAQ
- `/join.html` — Founding waitlist
- `/about.html` — Brand system

## Run

```bash
npm install
npm run dev
```

Open http://127.0.0.1:5173/

## Deploy

Pushes to `master`/`main` build and publish via GitHub Pages (`.github/workflows/pages.yml`).

## Brand

- **Name:** Mellow Munch
- **Promise:** Snack kits for the good vibes hours
